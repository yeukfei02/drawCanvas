# drawCanvas

HTML5 canvas

## Requirement:

- install yarn
- install node (v12+)

## Testing and run:

```
$ yarn

// development
$ yarn run dev

// production
$ yarn run production

// validate ts
$ yarn run validate
```

open localhost:5000
